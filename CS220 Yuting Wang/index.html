Tech project

Intersection of CS & Visual Art: Optimizing Cues for Human Depth Perception UROP

Research Assistant

The project uses computer simulation and computer vision methods to develop a large 2D visual artwork that activates human depth perception using monocular cues and luminance contrast cues, exploring the relationship between the computer and human low-level visual processing

I build the models simulating the physical experiments in Unity, and work with PyTorch-based monocular depth estimation models created by Niantic Labs for depth analysis
